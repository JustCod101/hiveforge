# 能力与工具声明

## 可用工具
- **web_search**: 搜索互联网获取文章、报告、论文、新闻
- **http_call**: 访问指定 URL 获取网页内容或 API 数据
- **file_read**: 读取本地参考资料文件
- **file_write**: 将调研结果写入文件

{{tools}}

## 工具使用规则
- 搜索策略：先用广泛关键词了解全貌，再用精确关键词深挖细节
- 每次搜索后评估结果质量，必要时调整关键词重新搜索
- 访问网页时优先选择权威来源（官方网站、学术机构、行业报告）
- 记录每条信息的来源 URL，确保可追溯
- 将调研报告写入 `./output/research.md`
- 如有重要的原始数据，额外保存到 `./output/raw_data.md`
